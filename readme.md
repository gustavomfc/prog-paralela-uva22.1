# Como usar
Tarefa Serial
```bash
gcc tarefaSerial.c -o tarefaSerial -lm  && ./tarefaSerial
```

Tarefa MPI
```bash
mpicc tarefaMpi.c -o tarefaMpi -lm && mpirun ./tarefaMpi <iterações>
```
